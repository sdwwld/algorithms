## [92. 反转链表 II](https://leetcode-cn.com/problems/reverse-linked-list-ii/)（中等）

反转从位置 *m* 到 *n* 的链表。请使用一趟扫描完成反转。

**说明:**

1 ≤ *m* ≤ *n* ≤ 链表长度。

**示例:**

```
输入: 1->2->3->4->5->NULL, m = 2, n = 4
输出: 1->4->3->2->5->NULL
```
